Welcome to the Henry github pages.

Henry is a Raspberry Pi Zero powered robot with a simple web interface for remote control. Henry was inspired by PiWars and the Zerobot feature in MagPi Magazine Issue 40 (Pi Zero launch edition)

Basic Robot Components

Henry uses most of the components featured in the zerobot article. Namely these are:

  • PiZero
  • 2 x 28BYJ-48 Stepper Motors
  • 2 x ULN2003A Motor Drivers

In addition to this Henry also uses:

  • 90 degree 40 pin header
  • Raspberry Pi WiFi Dongle
  • Henry desktop hoover
  • Motor compatible wheels
  • Mobile phone battery pack
  • various cables, nuts bolts etc.
